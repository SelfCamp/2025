You can totally do this!

Instructions: Use your arrow keys to move the tiles. When two tiles with the same number touch, they merge into one! The game's goal is to reach 2049 points. You will have one minute to figure out what to do when you reach 2048!
This is a practice project created by Adam Hosman and Oliver Juhasz. The game was built up from scratch by using Vanilla JS, based on the game "2048" created by Gabriele Cirulli. Curious about the code? Check out our repo here!
